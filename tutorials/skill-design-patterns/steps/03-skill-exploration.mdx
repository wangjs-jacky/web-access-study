# 实测验证：从 Prompt 到脚本的完整链路

| 你会怎么说 | 背后触发什么 |

## CDP 全流程验证

<CodeBlock language="bash" copyable>{`# 一键验证脚本
T=$(curl -s "http://localhost:3456/new?url=https://example.com" | grep -o '"targetId":"[^"]*"' | cut -d'"' -f4)
echo "创建 tab: $T"
echo "页面信息: $(curl -s "http://localhost:3456/info?target=$T")"
echo "页面标题: $(curl -s -X POST "http://localhost:3456/eval?target=$T" -d 'document.title')"
echo "截图: $(curl -s "http://localhost:3456/screenshot?target=$T&file=/tmp/web-access-test.png")"
echo "关闭: $(curl -s "http://localhost:3456/close?target=$T")"`}</CodeBlock>

## 交互能力速查

| 操作 | 命令 |
|------|------|
| JS 点击 | `curl -s -X POST ".../click?target=ID" -d 'selector'` |
| 真实鼠标 | `curl -s -X POST ".../clickAt?target=ID" -d 'selector'` |
| 文件上传 | `curl -s -X POST ".../setFiles?target=ID" -d 'json'` |
| 滚动 | `curl -s ".../scroll?target=ID&y=3000"` |

## CDP API 清单

| CDP 域 | 命令 | 对应 HTTP API |
|--------|------|-------------|
| Target | getTargets / createTarget | /targets / /new |
| Page | navigate / captureScreenshot | /navigate / /screenshot |
| Runtime | evaluate | /eval |
| Input | dispatchMouseEvent | /clickAt |
| DOM | setFileInputFiles | /setFiles |
